FLEX:2 & FLEX:1

Die Flexwerte addieren sich. Der Zwischenraum wird prozentual aufgeteilt
Flex:2 + Flex:1 = 3 (Flex:2 bekommt 66% und Flex:1 bekommt 33% des Zwischenraums)

.prarent{

display: flex;

justify-content: center;

align-items: center;

}

.childA{

flex:2;

}

.childB{

flex:1;
